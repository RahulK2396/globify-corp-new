# Globify — Design · Build · Scale

Corporate website for Globify, delivering custom web & mobile app development, e-commerce, ERP, AI automation, and digital marketing solutions.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Development

```sh
npm install
npm run dev
```

## Deployment

The site is published at [globify-corp.lovable.app](https://globify-corp.lovable.app)
